ABOUT

What it does

The model that can predict diamond prices based on various factors such as carat weight, cut, color, clarity, and other relevant characteristics

How i Build it

I developed a machine learning model using a Kaggle dataset and created an API to leverage its functionality. This API can be utilized to retrieve data for our website.

Explanation of the Technology

The diamond price forecasting model utilizes Random Forest Regression for predictive analysis and an API for seamless integration into web applications, enabling real-time access to predicted diamond prices.

Accuracy

Due to the extensive dataset, we've achieved an impressive accuracy level of 99.2%.
